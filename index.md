你可以根据自己的需要添加更多的网址，也可以更换封面图片。记得将这个 Markdown 文件命名为 `index.md` 并上传到你的 GitHub 仓库中。然后在仓库的设置中启用 GitHub Pages 功能，就可以通过访问 `https://yourusername.github.io/yourrepositoryname/` 来查看你的网址收藏夹了。
